# Subscription Management Web UI
Web Ui for the Subscription Management project [here](https://github.com/cjhetzle/PayPalProject).

## Installation

Use [Node.js](https://nodejs.org/en/) to install.

```bash
npm install 
npm install -g @angular/cli
```

## Usage
### Dev Server
```bash
ng serve Paypal
```
### Build Docker Image
```bash
docker build .
```
